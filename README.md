# 🍳 FoodieHub

**FoodieHub** is a modern, user-friendly recipe platform where food enthusiasts can discover, share, and organize their favorite recipes. Perfect for home cooks, aspiring chefs, or anyone looking for culinary inspiration!  

---

## 🚀 Features

- **Browse Recipes:** Search recipes by ingredients, cuisine, or dietary preferences.  
- **Save Favorites:** Bookmark recipes to access them later easily.  
- **Share Recipes:** Submit your own recipes and share them with the community.  
- **Meal Planning:** Organize weekly meals and plan ahead.  
- **Interactive UI:** Responsive and mobile-friendly design.  

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JS
- **Backend:** PHP, MySQL
- **Database:** PHPMyAdmin
